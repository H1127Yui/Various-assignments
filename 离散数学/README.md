# 作品描述

## 描述

 本程序是一个通用程序，可以解决图的矩阵输出图的问题，即输入一个简单图G的邻接矩阵或者距离矩阵，输出该图的图片。

## 使用的程序开发工具：

开发语言：Python3.11.2

开发环境：Visual Studio Code

## 必须的库

```
pip install numpy
```

```
pip install tkinter
```

```
pip install networkx
```

```
pip install matplotlib
```

## 运行方法说明：

在一台安装了Python3的电脑上双击运行文件“启动.bat”即可。
